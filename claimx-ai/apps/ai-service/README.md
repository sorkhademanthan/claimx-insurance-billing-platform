## Tech Stack
- **Language:** Python 3.9+
- **Framework:** FastAPI
- **Server:** Uvicorn

## Running Locally
```bash
# Install dependencies
pip install -r requirements.txt

# Run server
python main.py
```
Server runs on: `http://localhost:8000`
